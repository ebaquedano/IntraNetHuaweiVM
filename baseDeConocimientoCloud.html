# LÍMITES HUAWEI CLOUD
1 Podrán cotizar un máximo de 5 VMs.
2 Elastic Cloud Server (Cotizar únicamente las categorías (S3/S6/T6/)).
3 Almacenamiento Elastic Cloud Server:(High I/O / General Purpose).
4 Volumen Backup Service, solo se podrá cotizar el mismo para el servicio de Elastic Volume Service (Discos)

# LÍMITES GCP
1 Podrán cotizar un máximo de 5 VMs.
2 Compute Engine (Cotizar únicamente las categorías (E2/N2/N2D/T2D/T2A)).
3 Almacenamiento Compute Engine:(Zonal Standard PD / Zonal Balanced PD / Zonal SSD PD).

# LÍMITES AZURE
1 Podrán cotizar un máximo de 5 VMs.
2 Virtual Machine (Cotizar únicamente las categoría General Purpose - Series (A/B/D/DC)).
3 Almacenamiento Virtual Machine:(HDD Standard / Standard SSD / SSD Premium).

IMPORTANTE: No se podrán cotizar Servicios Profesionales, todo servicio entra por un proceso consultivo.

  • Capacitación Huawei

    • Para el uso de la calculadora de Huawei, debemos de tener en claro las consultas que debemos realizarle al cliente. Al cliente debemos realizarle algunas preguntas claves: ¿Qué cantidad de procesadores requiere en su VM?
    # Tipo de ECS (Elastic Cloud Server) Cantidad de procesadores
    1 General Computing-basic (T6) 1, 2, 4, 8, 16 vCPUs*
    2 General Computing (S6) 1, 2, 4, 8 vCPUs**
    3 General Computing-plus (C6s) 2, 4, 8, 12, 16 , 24, 48, 64 vCPUs***
    NOTA*: Intel® Xeon® Scalable Processor | Basic/Turbo frequency: 2.2 GHz/3.0 GHz
    NOTA**: 2nd Generation Intel® Xeon® Scalable Processor | Basic/Turbo frequency: 2.6 GHz/3.5 GHz
    NOTA***: 2nd Generation Intel® Xeon® Scalable Processor | Basic/Turbo frequency: 2.6 GHz/3.5 GHz

    ¿Qué cantidad de memoria Ram va a utilizar en su VM?
    # Elastic Cloud Server Cantidad de Memoria
    1 Memoria Ram 1, 2, 4, 8, 16, 32, 64, 128, 256 GB*
    NOTA*: La cantidad de memoria que va a figurar visible, va a depender de dos variables:
    1. El tipo de ECS: General Computing Basic/Computing/Plus
    2. La cantidad de procesadores que se selecciona.

    ¿Qué cantidad de almacenamiento? (Teniendo en cuenta que la cotización se compone de 2 discos:
    a. El primero es exclusivo para el SO.
    b. El segundo es para la VM general.
    # Almacenamiento ¿Cómo cotizar?
    1 Linux Se cotiza directamente desde la calculadora
    2 Windows* Se cotiza por fuera de la calculadora

    NOTA*: Revisar el spreadsheet de las licencias de Windows: Spreadsheet
